Event Details

Live A Hero’s long long long await Valentine event is here, introducing the game’s first tradable named character Hisaki. Event gacha summon includes a new character Flamier (Hero & Sidekick) and two time-limited variants “Active Agent” Kyoichi (Hero) and “Secret Agent” Shoen (Hero).

For this event quest there are one-time boost items you can use before entering each stage, these items dropped from the event quest itself which is:

  • Power Booster, increase team atk by 100 each for a maximum of 1000
  • Energy Tank, increase team HP by 200 each for a maximum of 2000
  • Defensive Shield, reduce received damage by 50 each for a maximum of 500
